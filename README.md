# Swedish for Leighcie 🐈

A little cat-themed website that teaches practical Swedish — greetings, getting
around, café & shop phrases, making friends, just-in-case basics, and (of
course) cat words. Tap any card to hear it spoken aloud, and there's a
"Feed the Cat" quiz to practice. Built as a birthday gift, made to grow.

**It's a plain website** — nothing to install to use it, just open the link.

## Want to change it?
You own this. The easiest way: open the folder in Claude and ask in plain
English ("add ten food words," "change the note up top," "make it more purple").
Everything you'd edit lives in **`data.js`**.

See **`CLAUDE.md`** for the full owner's guide and **`DEPLOY.md`** for how to
publish updates.

## Files
| File | What it is |
|------|------------|
| `index.html` | The page |
| `data.js` | All the words + the personal note — **edit this** |
| `styles.css` | Colors & styling (theme colors at the very top) |
| `app.js` | The machinery (cards, tap-to-hear, quiz) — leave alone |
| `CLAUDE.md` | Owner's guide |
| `DEPLOY.md` | Hosting + how to publish changes |
